②、队列（Queue）

Queue继承树

Queue本身是一种先入先出的模型(FIFO)，和我们日常生活中的排队模型很类似。根据不同的实现，他们主要有数组（ArrayList）和链表（LinkedList）两种实现形式

这里给出Queue与Deque的两种类中的操作

public interface Queue<E> extends Collection<E> {  
  
    boolean add(E e); // 添加元素到队列中，相当于进入队尾排队。  
  
    boolean offer(E e);  //添加元素到队列中，相当于进入队尾排队.  
  
    E remove(); //移除队头元素  
  
    E poll();  //移除队头元素  
  
    E element(); //获取但不移除队列头的元素  
  
    E peek();  //获取但不移除队列头的元素  
}

public interface Deque<E> extends Queue<E> {  
  //add的相关方法
 boolean add(E e);  
  
boolean offer(E e);  
  
void addFirst(E e);  
  
void addLast(E e);  
  
boolean offerFirst(E e);  
  
boolean offerLast(E e);
    //remove的相关方法
    E removeFirst();  
  
E removeLast();  
  
E pollFirst();  
  
E pollLast();  
  
E remove();  
  
E poll();
    //get的相关方法
    E getFirst();  
  
E getLast();  
  
E peekFirst();  
  
E peekLast();  
  
E element();  
  
E peek();
}

注意：其中add和remove是会抛出异常的。

二、三道栈与队列的应用（leedcode）

1、栈实现队列

①、题目：用两个栈实现一个队列。队列的声明如下，请实现它的两个函数 appendTail 和 deleteHead ，分别完成在队列尾部插入整数和在队列头部删除整数的功能。(若队列中没有元素，deleteHead 操作返回 -1 )

②、解题思路：Key:判断顺序，如果辅助栈S2中有元素，那么需要输出，如果没有那需要看S1中，入队都是入S1（主栈）。

③、代码：

class CQueue {
    Stack<Integer> S1 ;
    Stack<Integer> S2 ;
    //声明两个栈
    public CQueue() {
    
    S1 = new Stack<Integer> (); //主栈
    S2 = new Stack<Integer> (); //辅助栈
    
    }
    // 入队操作
    public void appendTail(int value) {
        S1.push(value);
    }
    // 删除操作
    public int deleteHead() {
        if(!S2.empty()) return S2.pop();
        else if(S1.empty()) return -1; 
        else{
            //将主栈的元素都放到辅助栈中
            
            while(!S1.empty()){
                int temp = S1.pop();
                S2.push(temp);
            }
            int  M =  S2.pop();
  
            return M;
        }
    }
}

/**
 * Your CQueue object will be instantiated and called as such:
 * CQueue obj = new CQueue();
 * obj.appendTail(value);
 * int param_2 = obj.deleteHead();
 */

2、返回栈的min

①、题目：定义栈的数据结构，请在该类型中实现一个能够得到栈的最小元素的 min 函数在该栈中，调用 min、push 及 pop 的时间复杂度都是 O(1)。

示例: 

MinStack minStack = new MinStack();

minStack.push(-2);

minStack.push(0);

minStack.push(-3);

minStack.min();  --> 返回 -3.

minStack.pop();

minStack.top();    --> 返回 0.

minStack.min();  --> 返回 -2.

②、解题思路：

思路1：

但是这样空间性能很差是O（n）；

思路二：每次压栈都把原来的最小值也压入。

③、代码：

思路一：

class MinStack {
    Stack<Integer> S1;
    Stack<Integer> S2;
    //声明两个栈
    /** initialize your data structure here. */
    public MinStack() {
        S1 = new Stack<Integer> ();  //主栈
        S2 = new Stack<Integer> ();  //辅助栈_1  （降序栈）
    }
    
    public void push(int x) {
        S1.push(x);                                         //入主栈
        if( S2.empty() || (x <= S2.peek())){                 //入降序栈
            S2.push(x);
        }
    }
    
    public void pop() {
        if( S1.pop().equals(S2.peek()) ){
            S2.pop();
        }
    }
    
    public int top() {
        return S1.peek();
    }
    
    public int min() {
        return S2.peek();
    }
}

/**
 * Your MinStack object will be instantiated and called as such:
 * MinStack obj = new MinStack();
 * obj.push(x);
 * obj.pop();
 * int param_3 = obj.top();
 * int param_4 = obj.min();
 */

思路二：

class MinStack {
    Stack<Integer> S;
    int min = Integer.MAX_VALUE;
    //声明两个栈
    /** initialize your data structure here. */
    public MinStack() {
        S = new Stack<Integer> ();  //主栈
    }
    
    public void push(int x) {
        
        if(x <= min) {
            //碰到更小的值，原来的最小值入栈
            S.push(min);
            min = x;
        } 
            //再将新的最小值入栈
        S.push(x);
    }
    
    public void pop() {
        //如果当前移除的值是最小值，弹出此值，然后min=原先的最小值
        if(S.pop()==min) min = S.pop();
    }
    
    public int top() {
        return S.peek();
    }
    
    public int min() {
        return min;
    }
}

/**
 * Your MinStack object will be instantiated and called as such:
 * MinStack obj = new MinStack();
 * obj.push(x);
 * obj.pop();
 * int param_3 = obj.top();
 * int param_4 = obj.min();
 */

1、栈实现队列

①、题目：

请定义一个队列并实现函数 max_value 得到队列里的最大值，要求函数max_value、push_back 和 pop_front 的均摊时间复杂度都是O(1)。
若队列为空，pop_front 和 max_value 需要返回 -1
示例 1：
输入:
[“MaxQueue”,”push_back”,”push_back”,”max_value”,”pop_front”,”max_value”]
[[],[1],[2],[],[],[]]

输出: [null,null,null,2,1,2]

示例 2：
输入:
[“MaxQueue”,”pop_front”,”max_value”]
[[],[],[]]

输出: [null,-1,-1]

限制：
1 <= push_back,pop_front,max_value的总操作数 <= 10000
1 <= value <= 10^5

②、解题思路：

对于普通队列，入队 push_back() 和出队 pop_front() 的时间复杂度均为 O(1)；本题难点为实现查找最大值 max_value() 的 O(1) 时间复杂度。

假设队列中存储 NN 个元素，从中获取最大值需要遍历队列，时间复杂度为 O(N) ，单从算法上无优化空间。

最直观的想法是 \维护一个最大值变量** ，在元素入队时更新此变量即可；但当最大值出队后，并无法确定下一个 \次最大值** ，因此不可行。

考虑利用 数据结构 来实现，即经常使用的 “空间换时间” 。如下图所示，考虑构建一个递减列表来保存队列 所有递减的元素 ，递减链表随着入队和出队操作实时更新，这样队列最大元素就始终对应递减列表的首元素，实现了获取最大值 O(1) 时间复杂度。

③、代码：

class MaxQueue {
    Queue<Integer> queue_min;           //主队列1
    Deque<Integer> deque;               //辅助队列(严格降序栈)  双端队列

    public MaxQueue() {
        queue_min = new LinkedList<>();
        deque = new LinkedList<>();
    }
    
    public int max_value() {
        return deque.isEmpty() ? -1 :deque.peekFirst();  //若主队列中没有元素，返回-1 若有元素，返回队头元素
    }
    
    public void push_back(int value) {
        queue_min.offer(value);
        while( ! deque.isEmpty() && value > deque.peekLast())       //当双端队列是空||队尾元素小于value，出队尾
            deque.pollLast();
        deque.offerLast(value);
    }
    
    public int pop_front() {
        if(queue_min.isEmpty()) return -1;
        if(queue_min.peek().equals(deque.peekFirst())) deque.pollFirst();;    
         return  queue_min.poll();   
          
    }
}

 * Your MaxQueue object will be instantiated and called as such:
 * MaxQueue obj = new MaxQueue();
 * int param_1 = obj.max_value();
 * obj.push_back(value);
 * int param_3 = obj.pop_front();
 */
